# AI Mortgage Intelligence Copilot

AI-powered mortgage analytics platform built using Python, Streamlit, OpenAI, LangChain, ChromaDB, and RAG.

## Features

- Upload HMDA mortgage datasets
- Analyze lending trends
- Compare loan types
- Generate AI-powered business insights
- Natural language mortgage Q&A

## Tech Stack

- Python
- Streamlit
- OpenAI API
- LangChain
- ChromaDB
- RAG
- GitHub
